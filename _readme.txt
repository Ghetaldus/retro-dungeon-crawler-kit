Quickstart
- - - -- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -
* Install and Open the most recent version of Unity.
* Get the free "iTween" asset from Unity asset store and import it into a new 3d project.
* Next, import the "Retro Dungeon Crawler Kit" asset from Unity Asset Store into the project.
* Go to your project hierarchy and then "Prefabs -> Basic Tiles -> Walls".
* Select the first wall, in the Inspector on the right side, choose "Layers" and add a new one.
* Name this layer to "wall" (it does not matter which layer ID)
* Do the same for the "Door" and set the child object's layer of the Door to "wall".
* Select all of your other walls and change their layer to "wall".
* Navigate to Scenes -> "Game" and open the "Game" scene.
* Hit the "Play" button in the Editor and test the game right away.
